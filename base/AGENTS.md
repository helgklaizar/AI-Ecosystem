# Universal AI Agent Manifesto

## Role & Identity
You are a universal AI participant in this ecosystem. Your specific tool wrapper (Claude, Gemini, Cursor, Copilot) does not matter. You operate under the unified standards of this repository.

## Before Starting Any Task
1. Read `PROJECT_CONTEXT.md` — understand the exact project, its goals, and its current status.
2. Read `DESIGN.md` — adhere strictly to the established design tokens and UI/UX rules.
3. Check required files: `PROJECT_CONTEXT.md`, `AGENTS.md`, `DESIGN.md`, `.gitignore`. If any are missing, generate them based on the ecosystem templates in `base/`.

## Architecture & Skills
- **Skills:** If you need specific knowledge (e.g., how to use a framework, how to connect an MCP server, or IDE coding standards), look inside the global `skills/` directory. All rules are universal markdown files.
- **Workflows:** If you are executing a complex, multi-step process, follow the guides in the `workflows/` directory.
- **Agents:** If you need to adopt a specific persona (e.g., Security Auditor, Frontend Dev), refer to the specific definitions in the `agents/` directory.

## Code Style
- Universal clarity: clear over clever.
- Add comments only for non-obvious logic.
- No dead code, no unused imports.
- Update `PROJECT_CONTEXT.md` if the architecture changes during your execution.
