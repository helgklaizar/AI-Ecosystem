# Google Gemini Reference Repositories

Этот навык дает доступ к официальным репозиториям от Google Gemini, которые мы склонировали локально для изучения и переиспользования кода, архитектуры и системных промптов в наших проектах.

## Доступные исходники

Все репозитории лежат тут: `~/.gemini/antigravity/skills/google-gemini-repos/repos/`

1. **`gemini-cli`** (`.../repos/gemini-cli`)
   - **Когда юзать:** Для построения и оптимизации терминальных агентов, понимания как Google структурирует логику вызова тулов и промптов.
   - **Суть:** Опенсорсный CLI-агент от Гугла.

2. **`computer-use-preview`** (`.../repos/computer-use-preview`)
   - **Когда юзать:** При попытках научить агента напрямую управлять компьютером (нажимать кнопки, делать скриншоты).
   - **Суть:** Экспериментальная имплементация computer use.

3. **`proxy-to-gemini`** (`.../repos/proxy-to-gemini`)
   - **Когда юзать:** Когда внешний софт зашит жестко на OpenAI и нам надо подменить бекенд на Gemini.
   - **Суть:** Прокси, мимикрирующий под OpenAI API.

4. **`cookbook`** (`.../repos/cookbook`)
   - **Когда юзать:** Как справочник-карманный-словарь рабочих примеров кода для Gemini API (в т.ч. интеграции LangChain, CrewAI, LangGraph).
   - **Суть:** Огромная коллекция гайдов и рецептов.

## Правила
- Я буду использовать `grep_search`, `list_dir` и `view_file` по этим папкам, чтобы доставать из них нужный код во время работы.
- Код в этих папках редактировать нельзя, они только "read-only" референс для меня.
