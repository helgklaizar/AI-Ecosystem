# Crash Detect (Crash Analyzer)

## Описание
Автономный итеративный дебаг рантайм-сбоев и плавающих ошибок. Инструмент для выявления причин падения без нарушения работы системы.

## Шаги
1. **Анализ стэктрейса**: Изучить предоставленные логи ошибки или воспроизвести падение локально.
2. **Гипотезы гибели**: Сформировать 3-4 наиболее вероятные причины дефекта.
3. **Бинарный поиск логами**: Расставить (по согласованию) временные `console.log`, счетчики или метрики в код, сужая зону падения.
4. **Перехват**: Перезапустить процесс, собрать доп. логи и найти строку/модуль-инициатор проблемы.
5. **Хотфикс**: Сгенерировать минимальный фикс с объяснением root cause.
