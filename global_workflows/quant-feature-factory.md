# /quant-feature-factory

## 📌 Описание
Полностью автоматизированный генератор альфы (Feature Engineering) для табличных и финансовых данных (`quant-laboratory`). Внедряет подход, при котором улучшение качества предсказаний (+20-50%) достигается за счет новых признаков, а не тюнинга модели.

## 🎯 Цели
1. Преобразовать сырые OHLCV-данные во множество технических и статистических индикаторов.
2. Автоматически отсеять шумовые признаки, оставив только те, что обладают реальной предиктивной силой.

## 🚀 Шаги
1. **Data Ingestion**:
   - Загрузка сырых временных рядов (OHLCV).
2. **Feature Generation**:
   - Накрутка индикаторов (RSI, MACD, Bollinger Bands, ATR).
   - Расчет временных лагов и роллингов (скользящих средних за N периодов).
3. **Feature Importance Rater**:
   - Обучение легковесной модели (например, Random Forest или Gradient Boosting с ранней остановкой).
   - Извлечение метрик `feature_importance`.
   - Отбрасывание 80% наименее значимых признаков.
4. **Export**:
   - Сохранение очищенного датасета с топ-20% фичей для финального бэктеста стратегии.

## 🛠 Инструменты
- `pandas-ta` или `ta` (Technical Analysis Library).
- `tsfresh` (для генерации фичей временных рядов).
